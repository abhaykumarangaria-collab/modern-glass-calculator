# 💎 Modern Glassmorphism Calculator

A sleek, high-performance web calculator featuring a modern **Glassmorphism** UI, responsive design, and an integrated calculation history sidebar.

![Calculator Preview](https://via.placeholder.com/800x450.png?text=Modern+Calculator+Preview) ## ✨ Features

-   **Glassmorphism UI:** Translucent frosted-glass effect with smooth CSS transitions.
-   **Calculation History:** A sliding sidebar that tracks your previous calculations with timestamps.
-   **Keyboard Support:** Fully functional using your physical keyboard (Numbers, Operators, Enter, Backspace, Esc).
-   **Responsive Design:** Works beautifully on desktops, tablets, and mobile phones.
-   **Precision Handling:** Solves the common floating-point math issue (e.g., 0.1 + 0.2 = 0.3).
-   **Smart Display:** Shows both current input and the previous operation for better context.
